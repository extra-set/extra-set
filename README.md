# set-extra

[![NPM](https://nodei.co/npm/set-extra.png)](https://nodei.co/npm/set-extra/)

Extra methods for [set].<br>
Browserified, minified module: [set-extra.min].

```javascript
const extra = require('set-extra');
// -> updated Set constructor
// Set.is(<value>)
// Set.equal(<set1>, <set2>)
// ...
```
